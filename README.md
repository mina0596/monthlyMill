## HTML

### component

공통 사용 컴포넌트 폴더

- header
- footer

### common

고객/메이커스 공통 화면

- login: 로그인
- main: 메인화면(시연 이후 사용)
- introduce: 월간방앗간 소개(=임시용 메인창)

### memberJoin

회원가입 관련 화면들

- join_method: 회원가입 첫 화면
- join_agreement: 약관동의창
- join_basic: 필수정보 입력
- join_additory: 추가정보 입력
- join_finish: 회원가입 완료

### customer

고객 이용 화면

- findItem: 떡 찾기
- cart: 장바구니
- payment:결제창
- payment_deposit: 결제계산창
- payment_confirm: 입금내역 확인
- loading: 결제대기, 매칭대기용 로딩 화면
- orderList: 나의 주문정보-주문배송조회
- orderList_cancel: 나의 주문정보-취소반품조회
- orderCancel: 주문취소창

### admin

관리자 화면

### makers

메이커스 화면
